== Description ==
WP-Polls (with CubePoints) is a modified version of [WP-Polls](http://wordpress.org/extend/plugins/wp-polls/ "WP-Polls") by Lester 'GaMerZ' Chan which works with the [CubePoints Plugin](http://cubepoints.com "CubePoints Plugin").

CubePoints is a point management system designed for WordPress blogs while WP-Polls is a plugin that allow users to create polls.

WP-Polls (with CubePoints) bridges these two plugins to allow registered users to earn points by voting in a poll.

The [CubePoints Plugin](http://cubepoints.com "CubePoints Plugin") is required for this plugin to work.
